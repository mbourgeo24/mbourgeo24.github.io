    Recursive binary search project

  • Recursive Binary Search exercise github files

    • This is a basic example of a Recursive binary search written in C++. A binary search algorithim has a complexity runtime of 0(Log n) which is much more efficient than the linear search. However a downside of the binary search is it must be sorted first. With modern computers normally unless a list is very large, a binary search algorithim is not needed.

    Notice with the recursive version of the search the first number and last number are both needed to otherwise overflow is very possible.
